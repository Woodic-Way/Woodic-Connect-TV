# 📺 WoodicConnectTv – V2Ray Client for Android TV

[![Android](https://img.shields.io/badge/Android-3DDC84?style=for-the-badge&logo=android&logoColor=white)](https://developer.android.com/)
[![API](https://img.shields.io/badge/API-21%2B-brightgreen.svg?style=flat)](https://android-arsenal.com/api?level=21)
[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://www.apache.org/licenses/LICENSE-2.0)

**WoodicConnectTv** is a lightweight and optimized V2Ray client designed specifically for **Android TV**. It provides fast and stable connections using the V2Ray core.

---

## ✨ Features

- 📡 **Protocol Support**: VMess, VLESS, Trojan, Shadowsocks
- 🔄 **Auto-Fetch Configs** from multiple public repositories
- 🧪 **Ping Testing** & automatic best-server selection
- 🧠 **Smart Caching** for faster reconnections
- 🎮 **TV-Optimized UI** with full D-pad/remote control support
- 🔒 **Secure Tunneling** via VPN
- 📊 **Real-time Status** display (connection state, speed, etc.)
- 🛡️ **Android 16+ Compatible** with proper foreground service handling

---

## 📱 Requirements

- Android TV device with **API 21 (Android 5.0)** or higher
- Internet connection

---

## 🚀 Installation & Setup

### Option 1: Download APK (Recommended)
1. Download the latest APK from the [Releases](https://github.com/your-username/WoodicConnectTv/releases) section.
2. Install the APK on your TV device (via USB or direct download).
3. Launch the app and press the **Connect** button.
